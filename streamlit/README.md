# Streamlit Apps

This directory contains Streamlit applications for visualizing NHL data.

## Getting Started

1. Install Streamlit dependencies:
```bash
pip install streamlit plotly snowflake-connector-python
```

2. Run the main dashboard:
```bash
cd streamlit
streamlit run nhl_dashboard.py
```

## Apps

- `nhl_dashboard.py` - Main player statistics dashboard
- `config.py` - Streamlit-specific configuration
