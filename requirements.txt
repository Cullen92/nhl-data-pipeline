# ==============================================================================
# NHL Data Pipeline - Local Development Dependencies
# ==============================================================================
# This file installs all dependencies needed for local Iceberg development.
# Dependencies are organized in pyproject.toml using optional dependency groups.
#
# For CI/CD: Only install dev tools with `pip install -e ".[dev]"`
# For local development: Install everything with `pip install -e ".[all]"`
#
# Dependency groups defined in pyproject.toml:
# - [project.dependencies]: Core dependencies (requests, pydantic, pandas, etc.)
# - [dev]: Development tools (ruff, pytest, mypy)
# - [iceberg]: Lakehouse dependencies (pyiceberg, duckdb, boto3, pyarrow)
# - [viz]: Visualization tools (streamlit, plotly)
# - [all]: All of the above
# ==============================================================================

# Install local package with all optional dependencies for full local development
